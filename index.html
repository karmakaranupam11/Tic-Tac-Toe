Tic Tac Toe

game rules:

  1. The game has a grid that's 3 squares by 3 squares.
  2. the player who is playing "X" always goes first.
  3. If a player is able to draw three Xs or three Os in a row, then that player wins.
  4. If all squares are filled and neither player has made a complete row of Xs or Os, then the game is a draw.
